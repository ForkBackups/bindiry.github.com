h3. Tools
* "Spine":http://esotericsoftware.com/ - 强大的跨平台骨骼动画编辑器
* "OgmoEditor":http://www.ogmoeditor.com/ - 基于 Adobe Air 的关卡编辑器
* "Gleed2D":http://gleed2d.codeplex.com/ - 一款通用2D关卡编辑器
* "RUBE":https://www.iforce2d.net/rube/ - 一款 Box2D 编辑器，可以生成JSON格式数据
* "Cadet":http://unwrong.com/cadet - 同样是一款物理引擎编辑器，支持2D和3D，2D主要针对Box2D
* "Particle Designer":http://www.71squared.com/en/particledesigner - 一款粒子编辑器，支持 Cocos2d、Starling等框架
* "DrawScript":http://drawscri.pt/ - 根据矢量图生成各语言代码
* "shoebox":http://renderhjs.net/shoebox/ - 一款游戏开发资源整理工具，支持目前大多数主流游戏框架

h3. Cross-Platform Game Development
* "Gideros Mobile":http://www.giderosmobile.com/ - 小巧，小手快速，使用lua编写代码的手机游戏开发平台
* "Moai":http://getmoai.com/ - 面向专业游戏开发者的开源手机游戏开发平台
* "Flambe":https://github.com/aduros/flambe 
  Flambe is an engine and asset pipeline for cross-platform multiplayer games.
  Written in Haxe, games are compiled to Flash and HTML5, with support for mobile devices. Server-side logic compiles to JS and runs on Node.js, for games that require multiplayer.
* "Dreemchest":http://dreemchest.com/
* "stencyl":http://www.stencyl.com/ - 傻瓜游戏制作工具

h3. Server
* "Chilly":http://chillyframework.com/ - 开源，同样基于 node.js 的游戏服务端框架

* "Firefly":http://firefly.9miao.com/ - 免费、开源、稳定、快速扩展、能 “热更新”的分布式游戏服务器端框架，采用Python编写，基于Twisted框架开发
* "Photon":http://www.exitgames.com/ - 跨平台的多人在线游戏服务端框架
* "Pomelo":http://pomelo.netease.com/ - 开源、快速、扩展性好、分布式游戏后端开发框架，网易出品
* "SmartFox":http://www.smartfoxserver.com/ - 比较有名的获平台多人在线游戏服务端框架
