h3. Network
* "shadowsocks":http://shadowsocks.org/
** "Shadowsocks-libev":https://github.com/madeye/shadowsocks-libev - shadowsocks 协议的轻量级实现
** "shadowsocks-go":https://github.com/shadowsocks/shadowsocks-go - 用 Go 实现的 Shadowsocks 客户端，支持多平台
** "shadowsocks-go-local":https://code.google.com/p/shadowsocks-go/downloads/list - 编译好的 shadowsocks-go 客户端
* "cow":https://github.com/cyfdecyf/cow - 一款 http 代理工具，支持 socks5 和 shadowsocks 做为二级代理
* "CFW":https://github.com/BB9z/CFW - 一份基于IP地址的白名单列表
* "GFW_Whitelist":https://github.com/n0wa11/gfw_whitelist
* "Flora_Pac":https://github.com/Leask/Flora_Pac - 一个以国内IP为白名单的PAC列表， "介绍文章":http://www.leaskh.com/blog/2013/10/30/%E6%B5%8B%E8%AF%95%E4%B8%BA-flora_pac-%E6%8F%90%E4%BE%9B-http-host/
* "xunlei-lixian":https://github.com/iambus/xunlei-lixian - 一款迅雷离线下载脚本
** "lixian-portal":https://github.com/layerssss/lixian-portal - 基于xunlei-lixian的一个UI实现

h3. Other
* "FastDFS":http://code.google.com/p/fastdfs/ - 一款高性能分布式文件系统
* "Phabricator":http://phabricator.org/ - Facebook 推荐出的基于web的项目管理系统，包含Code Review、工作流管理、漏洞追踪等功能
* "Gerrit":http://code.google.com/p/gerrit/ - Google的基于Web的项目Code Review系统
* "codeswarm":https://code.google.com/p/codeswarm/ - 以图形动画形式显示版本管理工具的提交过程
* "QR Code Generator":http://d-project.googlecode.com/svn/trunk/misc/qrcode/index.html - 二维码生成类库，支持众多语言
